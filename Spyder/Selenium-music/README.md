
修改自 [here](https://github.com/Denon/syncPlaylist) 一个 18年的失修项目. 现在的逻辑很简单: 有一个歌单, 用 Selenium 模拟搜索和加入歌单.

实际上有更简易的方案: [歌单助手](https://music.wechatsync.com/explore/#/transfer) 用于从歌单 list 导入到网易云. 这里简单调试了一下 Selenium, 少许成就感.
